# Omni VS Code Support

To enable syntax highlighting and language support for Omni:

1. Open a terminal (PowerShell or Bash).
2. Navigate to the project root.
3. Run the following command:
   ```powershell
   code --install-extension .\omni-vscode
   ```
4. Restart VS Code if prompted.

Alternatively, you can copy the `omni-vscode` folder to your VS Code extensions folder:
- Windows: `%USERPROFILE%\.vscode\extensions`
- macOS/Linux: `~/.vscode/extensions`ipp appliedipp
